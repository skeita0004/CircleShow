# 円表示プログラム (ネットワーク付き)

## 概要

サーバ *1 対 クライアント *Any で通信をするプログラム。

## 環境

|箇所|内容|
|-:|:-|
|ライブラリ?|WinSock2|
|人数|4人|
|開発開始日|2025/12/05|
|開発期間(予定)|2025年内|

### ライブラリ

- DxLib (クライアントのみ)
- WinSock2

## 目的

- サーバクライアント方式のオンラインゲームをメンバ一人ひとりが作れるようになること。
- メモリ上の構造を理解するために、構造体をバイト配列として分解できること。
- メンバ全員が全体を見ることができ、全体のどの部分を作っているか意識できるチーム制作にすること。

## ブランチ

- `master` : ドキュメントを書いていくブランチ
- `dev-serv` : サーバサイドのプログラムを組んでいくブランチ
- `dev-cl` : クライアントサイドのプログラムを組んでいくブランチ
